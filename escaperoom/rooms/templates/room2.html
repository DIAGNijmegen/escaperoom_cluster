Assignment 2

Great work on the first assignment! Now, let's move to the next challenge.

Your task is to kick off a interactive job using the Docker image you created in Assignment 1 and run a Jupyter notebook in VScode on the SOL2 cluster. Damn, that's a lot of buzzwords. You got this!
  • Use the Docker image you pushed in Assignment 1.
  • Mount the directory where the training data is stored: /data/pathology.
  • Start the job with:
    • Using sbatch and attaching the running container to VSCode via SSH
  • Run the data.ipynb notebook, which does some basic data analysis and outputs a label distribution plot.
  • Download data.ipynb

Tips

If you're having trouble with this assignment, check out this page.
Daan made a package called deepops-slurmjob that creates and launches interactive jobs for you! Note: You will need an additional Python package that is not in the Docker image. You can simply use pip install to add it.

To acquire your puzzle piece

The assignment is passed once you successfully run the data.ipynb notebook and push the label distribution plot to your team Github repository. Create an issue, mention Clement (clemsgrs) and Daan (daangeijs), and include the SHA of your commit in the issue. They will review your repository and provide you with the next puzzle piece in your Issue.
