Assignment 1

Okay you made it to the first assignment. Good job!

Your goal is to create a Dockerfile, build a Docker image and push it to the registry.
  • We provide you with a template Dockerfile and a list of Python packages that need to get installed. Make sure you update the Dockerfile and that you install the packages listed in the requirements.txt file in your Docker image.
  • Build the docker image and push it to the registry.
  • Download Dockerfile
  • Download requirements.txt

Tips

Trouble building a Docker image? Check out the Docker build documentation.
Trouble pushing a Docker image? Check out the Docker push documentation.
Trouble installing Python packages? Check out the pip install documentation.
Is it not working on your local machine? Try to build and push the Docker image on the SOL2 cluster using dlc-duglet

To acquire your puzzle piece

The assignment is passed once you push the updated Dockerfile to your team Github repository. Create an issue, mention Clement (clemsgrs) and Daan (daangeijs), and the SHA of your commit in the issue. They check your repository and give you the puzzle piece by putting it in your Issue.
